**Chester**
The Chester program is a minimax based chess engine created using assembly 8086.
It can be ran on DOSBOX with the necessary run files.
The engine can calculate about 3 moves forward in reasonable time.

When executing the program a chess GUI will be opened, the human is down and Chester is above, 50% chance to start as white.

**Game**
A 1v1 game made in c# player1 using arrows and player2 WASD.
